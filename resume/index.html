数风科技(深圳)有限公司 （ 2019年06月 ~ 2019年09月）

FileStorm (未上App Store,企业签名)

项目简介

FileStorm 是款存储手机客户端，是对FileStorm分布式存储平台的一个更新与升级，提供基础的文件存储与检索服务，以及挖矿功能。

技术要点

  • 采用Swift作为主要开发语言进行独立开发,同时使用MVC 设计模式, xib + SnapKit 快速搭建UI;

  • 使用公有库YHKit,对项目精简,以及快速开发;

深圳趣拼云科技有限公司 （ 2018年07月 ~ 2019年04月）

趣拼

项目简介

2018年7月份参与趣拼iOS客户端开发, 趣拼作为一个新星购物电商平台,主打拼字, 半价,有奖,普通,三种拼团玩法,当然也有正常购物模式,对接京东商品。

作为公司核心程序员,我不但完成了iOS客户端编码工作,提出了移动端组件化。通过组件化有效提高了代码的可维护性, 扩展性,解决部分业务耦合,使业务更加清晰。

技术要点

  • 采用Swift作为主要开发语言进行独立开发,同时使用MVC 设计模式, xib + SnapKit 快速搭建UI;
  • 使用YHKit 拆分基类分类;
  • 部分业务制作cocoapods私有库对业务进行组件化,拆分项目结构;
  • 采用Moya + Alamofire + RxSwift + ObjectMapper 网络请求及解析服务器数据;
  • 多target配置, 部署develop,preview,release 环境, 解决测试人员同时测试三套环境问题,提升测试速度,方便各版本之间对比;
  • 使用fastlane 结合多target,部署到蒲公英方便提测,提升开发效率;
  • 利用Swift Protocol Extension特性, 解决了从view 从xib中加载时需要强转,及部分模块的解耦。

广州搏丽通软件科技有限公司 （ 2016年02月 ~ 2018年05月 ）

联商客

联商客是一款电商 的App。

技术要点

  • 基于Objective-C 开发, 使用Swift 进行重写, 两种语言进行混编开发,同时使用MVC 设计模式和同事协作开发;
  • 重写项目改用Moya + Alamofire + RxSwift + ProtocolBuffer 进行网络请求及解析;
  • 使用蒲同英部署方便应用测试;
  • 集成高德SDK,使用LBS显示附近联盟商家。

项目简介

以上几款业务基本相似,以下简称App,该款App P2P 投资理财类, 充值买入一定数量的积分,每天返还积分,三款模式大同小异,各有千秋。

技术要点

  • 基于Objective-C 开发,同时使用MVC 设计模式和同事协作开发;
  • 采用AFNetworking 网络请求,MJExtension进行JSON解析,二次封装方便扩展和调用;
  • 采用SDWebImage 下载并显示图片到cell;
  • 集成高德SDK,使用LBS显示附近联盟商家,并进行路径规划和导航;
  • 共同讨论线下支付流程,模仿支付宝扫码支付流程,实现线下支付;

  • 使用融云SDK 实现聊天模块
